# Stock Management System

Minimal WinForms stock management example (Products, StockMain, Form1).

Prerequisites
- Windows
- Visual Studio (e.g. Visual Studio Community 2026)
- .NET Framework 4.7.2

Open and run
1. Open Visual Studio and choose `Open -> Project/Solution` or `Open -> Folder` and select the repository root.
2. Build the solution (`Build -> Build Solution`).
3. Run the app (`Debug -> Start Debugging` or `Start Without Debugging`).

Quick Git commands

Important files
- `Products.cs`, `Products.Designer.cs` – product form
- `StockMain.cs`, `StockMain.Designer.cs` – main form
- `Form1.cs` – additional UI
